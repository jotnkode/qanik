## About the project 
qanik is a simple rust implementation of twitters SnowFlake.
### Built with
* rustc 1.83.0

### Getting started
Add qanik to your cargo dependencies.</br>
Ex:</br>
```
[dependencies]
    
qanik = "0.1.0"
```
